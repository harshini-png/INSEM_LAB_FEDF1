# Login and Registration Page

A simple Login and Registration Page developed using HTML, CSS, and JavaScript. This project demonstrates basic form creation and client-side validation.

## Features

- User Registration Form
- User Login Form
- JavaScript Form Validation
- Simple and User-Friendly Interface
- Easy to Customize

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Registration Fields

- First Name
- Last Name
- Email
- Password
- Region
- Date of Birth
- Phone Number

## Login Fields

- Username
- Password

## Project Structure

```
project-folder/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Run `index.html` in any web browser.
4. Register a new account.
5. Login using the provided credentials.

## Validation

- All fields are required.
- Empty fields are not allowed.
- Success messages are displayed after validation.
